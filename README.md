## What is it?

Credit Optimizer is a **Manus AI Skill** that reduces credit consumption by **30-75%** without any quality loss. It works by:

1. **Intelligent Model Routing** — Routes tasks to Standard mode when MAX isn't needed (~60% of tasks)
2. **Context Hygiene** — Prevents "context rot" that causes repeated work after iteration 8-10
3. **Prompt Decomposition** — Breaks monolithic prompts into optimally-routed sub-tasks
4. **Smart Testing** — Tests code section-by-section instead of full rebuilds
